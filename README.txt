# Embetty

Embetty displays remote content like tweets or YouTube videos without
compromising your privacy.

It consists of two components:

1. A client side script
   Download the latest release from https://github.com/heiseonline/embetty
   and copy the extracted embetty.js file to /libraries/embetty/embetty.js

2. A server side service
   Get and run your instance of https://github.com/heiseonline/embetty-server

## Installation
Like any other module, for help visit:
https://www.drupal.org/docs/8/extending-drupal-8/installing-drupal-8-modules

Then download the embetty release and extract it.

## Configuration
After installing the module got to the configuration page and provide the server
details at /admin/config/services/embetty
